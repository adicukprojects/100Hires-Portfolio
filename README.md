# 100Hires Portfolio Project

## Tools Installed

- Cursor IDE — AI-powered code editor
- Claude Code extension — Anthropic's AI coding assistant
- Codex extension — OpenAI's AI coding tool

## Steps Completed

1. Downloaded and installed Cursor IDE on Windows
2. Opened Cursor editor and installed Claude Code extension via Extensions panel (Ctrl+Shift+X)
3. Installed Codex extension in Cursor
4. Created a public GitHub repository named 100hires-portfolio
5. Cloned the repository into my local machine using Git: Clone in Cursor
6. Opened the repository folder in Cursor editor
7. Updated README.md with project documentation

## Issues Encountered & How I Solved Them

- Git was not installed initially, which prevented cloning — solved by downloading and installing Git from git-scm.com
- Cursor dashboard opened in browser instead of the desktop app — solved by launching the installed desktop application from the Start menu
